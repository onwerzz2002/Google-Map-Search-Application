
1. You can input the address in the search bar with placeholder "Enter place", and select Place Type and Search Range. Then 
   you can go on your search.

2. After you finish your place search, it will show a list of locations with the specified place type within a certain range
   choosen by you.

3. Once you move mouse over the marker on the map, it will change its color to help you see it more clearly. In addition,
   a list of name of will be shown on the left part underneath the Result box. Similarly, once you mouse over the name of place
   show on the list, not only the name on the list changes its color but also the corresponding arrow changes its color and size
   on the map.

4. Now, you can do the next search~

Techniques: HTML/CSS/JavasScript/JQuery/Google Map API
